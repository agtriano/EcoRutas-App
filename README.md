# 🌿 EcoRutas: Gasolineras Baratas y Rutas Eficientes

![Estado de la App](https://img.shields.io/badge/Estado-En_Producci%C3%B3n-brightgreen)
![Plataforma](https://img.shields.io/badge/Plataforma-Android-blue)

**EcoRutas** es la solución inteligente para conductores que buscan ahorrar en cada kilómetro. Nuestra aplicación te ayuda a encontrar las gasolineras más económicas cerca de tu ubicación y a planificar rutas sostenibles.

---

## 🚀 Características Principales

* **Ahorro Real:** Comparativa de precios de combustible en tiempo real.
* **Rutas Optimizadas:** Algoritmos que calculan el trayecto más corto y ecológico.
* **Interfaz Intuitiva:** Mapas claros y navegación paso a paso.
* **Actualización Constante:** Precios verificados diariamente.
